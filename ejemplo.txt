pwd - Te indica en donde estas ubicado actualmente 
cd NombreDeLaCarpeta - Navegar internamente a una carpeta 
cd .. -Salir fuera de la carpeta actual 
mkdir NombreDeLaCarpeta- crear una carpeta nueva

3) git commit -m "Primer commit,agregando archivo de ejemplo.txt(mensaje para el commit"
4) git remote add url-del-github
5) git push -u origin master git push -u origin main(solo la primera vez )
